# Titanic Survival Analysis Dashboard

This project explores passenger survival patterns from the Titanic dataset using a visual dashboard built with Power BI. The aim of the analysis is to understand how different factors such as passenger class, gender, fare, and embarkation port influenced survival outcomes.

## Project Overview

The Titanic disaster is one of the most well-known maritime tragedies in history. By analyzing passenger data, we can uncover patterns that affected the chances of survival.

In this project, the dataset was first cleaned and prepared using Microsoft Excel. After cleaning the data, Power BI was used to build an analytical dashboard to visualize key survival patterns.

## Tools Used

* Microsoft Excel (Data Cleaning)
* Power BI (Data Visualization)
* Data Analysis

## Key Metrics

* Total Passengers: 418
* Average Age: 30.27
* Total Survivors: 152
* Maximum Age: 76
* Passenger Classes: 3
* Highest Fare: 1K

## Key Insights

* The overall passenger survival rate was **36.36%**.
* **Female passengers had a significantly higher survival rate** compared to male passengers.
* **Southampton recorded the highest number of survivors** among the embarkation ports.
* **First-class passengers paid the highest fares** and had better survival outcomes.
* Passenger class played an important role in survival probability.
